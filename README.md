# Live AI Vedic Astrologer Chat - Landing Page

A premium, highly-converting SaaS landing page for an AI-powered Vedic astrology chat platform. This design utilizes modern UI/UX principles, including glassmorphism, soft cosmic gradients, and smooth scroll animations.

## Features
- **Fully Responsive**: Adapts seamlessly to desktop, tablet, and mobile devices.
- **Glassmorphism UI**: Beautiful frosted-glass effect for pricing and feature cards.
- **Interactive Pricing Selector**: Dynamic pricing updates using Vanilla JavaScript.
- **Mock Chat Interface**: A visual representation of how the AI chat works.
- **Accordion FAQ**: Built-in script for expandable/collapsible questions.
- **SEO Optimized**: Includes schema markup (JSON-LD) for Products and FAQs.

## Tech Stack
- **HTML5** (Semantic structure)
- **CSS3** (Custom properties, Flexbox, CSS Grid, Keyframes)
- **Vanilla JavaScript** (DOM manipulation, IntersectionObserver for scroll animations)
